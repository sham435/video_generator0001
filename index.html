What Video Generator Does

Smart Prompt Generation

Enter a subject and outfit list. The AI generates structured video prompts with camera specs, lighting, transitions, and shot-by-shot storyboards — ready for any AI video model.

Multi-Style Social Posts

From one prompt, automatically creates 7 social media post styles: casual swipe, high-energy grind, editorial thought-leadership, behind-the-scenes, results analytics, and carousel plans.

AI Video Rendering

Renders videos using NVIDIA NIM cloud AI (FLUX.1-dev for images, with video model support). No local GPU required — runs entirely in the cloud.

Automated Publishing

One-click publish to TikTok, LinkedIn, and Instagram. The pipeline handles formatting, media attachment, and scheduling via n8n workflow automation.

News-to-Video Pipeline

Automatically fetches trending headlines every 30 minutes, generates video prompts and social posts for each, and publishes them — fully hands-free.

Sandboxed Execution

All video rendering runs in isolated Docker containers with resource limits, privilege drops, and automatic cleanup — secure by design.

How It Works

1. Describe Your Content

Tell the app who or what the video is about, choose outfits/scenes, and select a post style. The AI generates a complete content package.

2. Render with AI

The prompt is sent to NVIDIA's cloud AI models to generate high-quality images or videos. Each render produces a vertical 9:16 format optimized for mobile.

3. Publish Everywhere

Generated content and media are published simultaneously to TikTok, LinkedIn, and Instagram through secure API integrations.

TikTok Integration

Video Generator uses the following TikTok products and API scopes:

Products Used

Login Kit

Used for secure user authentication. When a user taps "Login with TikTok," they authorize the app via TikTok's OAuth flow. This grants the app an access token used to call other TikTok APIs on the user's behalf.

Content Posting API

Used to upload and publish AI-generated videos directly to the user's TikTok feed. The app renders a video, uploads it to TikTok's servers, and the user confirms before it goes live.

Display API

Used to read the user's profile information and display it within the app, confirming which TikTok account is connected.

Scope Details

user.info.profile

Read access to the user's profile information including profile_web_link, profile_deep_link, bio_description, and is_verified. Used to display the connected TikTok account's name and avatar within the app interface, so the user can confirm which account will receive published content.

user.info.stats

Read access to a user's public statistics such as likes count, follower count, following count, and video count. Used to display account analytics within the app, helping users understand their audience reach before publishing content.

video.list

Read a user's public videos on TikTok. Used within the app to display the user's existing video library, allowing them to reference previous content when generating new videos and avoid duplicate topics.

video.upload

Upload video files to TikTok's servers. After the AI renders a video using NVIDIA's cloud, the app uploads the resulting video file to TikTok via the Content Posting API. The video is stored temporarily on TikTok's servers pending the user's publish confirmation.

video.publish

Publish an uploaded video to the user's TikTok feed. After upload, the user reviews and confirms the video within the app, and this scope finalizes the publishing action. The video then appears on the user's TikTok profile and in their followers' feeds.
